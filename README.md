Aplikasi konsol Java sederhana untuk menghitung Total Gaji Karyawan berdasarkan Golongan dan Jam Lembur yang dimasukkan.

Deskripsi Proyek:

Program ini berfungsi untuk menghitung gaji total seorang karyawan dengan menjumlahkan Gaji Pokok dan Gaji Lembur.

Gaji Pokok ditentukan berdasarkan golongan karyawan (A, B, atau C).

Gaji Lembur dihitung sebagai persentase tertentu dari Gaji Pokok, tergantung pada jumlah jam lembur.

Dibangun Dengan:

Bahasa Pemrograman: Java

Lingkungan Pengembangan (IDE): Netbeans
